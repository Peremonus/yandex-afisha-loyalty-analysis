# Анализ лояльности пользователей Яндекс Афиши

## Цель
Выявить факторы, влияющие на повторные покупки, и предложить рекомендации по повышению удержания клиентов.

## Данные
- Период: июнь – октябрь 2024
- Таблицы: `afisha.purchases`, `afisha.events`, `afisha.city`, `afisha.regions`
- Основные поля: `user_id`, `order_id`, `revenue`, `tickets_count`, `device_type_canonical`, `event_type_main`, `service_name`, `region_name`

## Структура репозитория
- `ticket_market.ipynb` – ноутбук с полным анализом
- `user_profiles.csv` – профили пользователей
- `requirements.txt` – зависимости
- `.gitignore` – исключённые файлы
- `README.md` – этот файл

## Запуск
1. Установить зависимости: `pip install -r requirements.txt`
2. Запустить Jupyter Notebook: `jupyter notebook ticket_market.ipynb`
3. Выполнить ячейки последовательно.

## Основные результаты
- **61,5%** пользователей совершили повторные покупки.
- Самые лояльные сегменты: **выставки** (64,4%), **театр** (63,8%), **десктоп** (64,1%).
- Оптимальный ценовой диапазон: **500–750 руб** – повторные пользователи здесь в 2 раза чаще.
- Покупка **2–3 билетов** даёт **74,2%** возвратов.
- **Регулярные покупки** (интервал 1–2 недели) и **суббота** – ключевые временные факторы.
- **Поведенческие признаки** (время жизни, интервалы, количество билетов) сильнее связаны с лояльностью, чем демографические.

## Рекомендации
- Фокусироваться на лояльных сегментах (выставки, театр, десктоп).
- Стимулировать регулярность покупок (напоминания, персонализированные предложения).
- Продвигать средний ценовой диапазон (500–750 руб) как зону высокой лояльности.
- Внедрить программы лояльности для групповых покупок (2–3 билета).
- Реактивировать пользователей с длительными перерывами (>30 дней).
- Использовать **медианную выручку** для мониторинга – она стабильнее средней.
